Sayaw ng mga kamay

Home / Educational/quiz Application

Project Details

Sayaw ng mga kamay is a mobile application designed to promote and teach Filipino Sign Language (FSL). The app provides lessons to help users learn FSL effectively, with visual demonstrations and a quiz feature that unlocks upon completing all lessons to reinforce learning and track progress.

The App was developed by a team of two developers.

  • Role: Visual Designer, UI
